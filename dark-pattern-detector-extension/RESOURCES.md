# Safe Web - Resources & References

## Dark Pattern Taxonomy
- **Deceptive Design (formerly Dark Patterns)**: https://www.deceptive.design/types
  - Official taxonomy of 16 dark pattern types
  - Maintained by Harry Brignull (coined "dark patterns" in 2010)
  - Used as the basis for our 112 example phrases

## Academic References
- [AidUI](https://arxiv.org/abs/2303.06782) - Dark pattern detection using CV + NLP
- [UIGuard](https://arxiv.org/abs/2301.09378) - Mobile app dark pattern detection

## Similar Projects
- [DarkSurfer-Extension](https://github.com/Venkateeshh/DarkSurfer-Extension) - BERT-based detection
- [Dark Pattern Detector](https://chrome.google.com/webstore/detail/dark-pattern-detector) - Various Chrome extensions
